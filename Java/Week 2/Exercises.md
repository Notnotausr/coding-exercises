# Week 2 Exercises
## Exercise 1: FizzBuzz
##### Requires knowledge of loops and conditionals.
### Requirements
Make a program that outputs the first 100 numbers with the following conditions:

1. When a number is divisible by 3, print "Fizz".
2. When a number is divisible by 5, print "Buzz".
3. When a number is both divisible by 3 and 5, print "FizzBuzz".
4. If none of the above conditions apply, print the number itself.
